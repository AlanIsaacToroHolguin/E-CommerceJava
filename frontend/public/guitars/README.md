## Recommended specs

- Format: **PNG** or **JPG** (PNG preserves transparency if you removed the background)
- Resolution: **800×800 to 1200×1200** square works best for the card grid
- File size: keep each under **300 KB** if possible (use a tool like https://squoosh.app to compress)
- Background: dark/moody fits the Solar-inspired theme nicely

## Where to find them

- The **manufacturer's website** (Fender, Gibson, Ibanez, etc.) — they publish high-res product photos
- **Reverb.com** product listings
- **Sweetwater.com** product galleries
- Stock sites: **Unsplash**, **Pexels**, **Pixabay** (search: `electric guitar`, `les paul`, `stratocaster`)

> ⚠️ For a portfolio project this is fine. For a real commercial store, get
> proper rights or shoot your own photos.
