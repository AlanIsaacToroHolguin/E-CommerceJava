## How to capture on Windows

1. Press **Win + Shift + S** → drag to select an area
2. The capture goes to your clipboard
3. Open Paint or any image editor → paste → save here with the exact filename

## How to record the GIF

Use [ScreenToGif](https://www.screentogif.com/) (free, portable, no install required):

1. Open it → click **Recorder**
2. Drag the capture window over the browser
3. Hit Record, navigate the demo flow:
   - Login (as Customer)
   - Storefront → click on a guitar → Add to cart
   - Open cart → adjust quantity → Proceed to Checkout
   - Place order → My Orders shows the new order
4. Stop, edit out dead frames, export as GIF
5. Save here as `00-demo.gif`

Once these files exist, the main `README.md` images render automatically — no extra wiring needed.
